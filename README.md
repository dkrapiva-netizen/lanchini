# LANCHINI — Интернет-магазин подарочных чайных пар

## О проекте

**LANCHINI** — интернет-магазин подарочных чайных пар (чашка + блюдце) в красивой упаковке.

**Целевая аудитория:** люди 30–50 лет, которые ищут красивые подарки на праздники — день рождения, юбилей, свадьбу, новоселье.

**Вдохновение по дизайну:** [KuchenLand](https://www.kuchenland.ru) — минимализм, белый фон, элегантные шрифты, качественные фото.

## Технологии

- HTML5
- CSS3 (адаптивный дизайн, CSS Grid, Flexbox)
- JavaScript (ES6+, без фреймворков)
- Google Fonts (Cormorant Garamond + Inter)
- LocalStorage для корзины
- Telegram Bot API для получения заказов

## Страницы и функционал

### Главная страница
- Шапка с логотипом, навигацией, поиском и иконкой корзины
- Слайдер-баннер (3 слайда с автопрокруткой)
- Секция «Популярные товары» (4 карточки)
- Секция «Почему выбирают нас» (4 преимущества)
- Подвал с контактами и ссылками

### Каталог
- 13 товаров в виде сетки
- Категории: **Все товары / Для женщин / Для мужчин** (вкладки)
- Сортировка: по умолчанию, по цене, по названию
- Фильтр по цене (от / до)
- Поиск по названию и описанию

### Товары по категориям

**Для женщин (9 товаров):**
Императорская роза, Голубая дымка, Сакура, Прованс, Версаль, Весенний луг, Пионы, Жемчуг

**Для мужчин (5 товаров):**
Золотой орнамент, Минимализм, Лофт, Самурай, Арктика

### Страница товара
- Название, цена, описание
- Таблица характеристик (материал, объём, размеры, упаковка)
- Выбор количества + кнопка «В корзину»
- Блок «Вам также понравится»

### Корзина
- Список товаров с названием и ценой
- Изменение количества и удаление
- Итого с расчётом доставки (бесплатно от 5000 ₽)
- Кнопка «Оформить заказ»

### Оформление заказа
- Форма: имя, телефон (с валидацией), email
- Адрес: город, улица, индекс
- Выбор способа доставки (курьер, СДЭК, Почта России)
- Комментарий к заказу
- **Отправка заказа в Telegram** — владелец получает уведомление мгновенно
- Страница «Спасибо за заказ» после успешной отправки

### Информационные страницы
- **О нас** — история бренда, ценности
- **Доставка и оплата** — способы, сроки, стоимость
- **Контакты** — телефон, email, адрес, форма обратной связи

## Интеграция с Telegram

Заказы поступают напрямую в Telegram-бот владельца магазина.
Уведомление содержит: имя и телефон покупателя, адрес доставки, состав заказа, итоговую сумму.

Настройка: в файле `pages/checkout.html` указаны `TG_TOKEN` и `TG_CHAT_ID`.

## Особенности

- **Адаптивный дизайн** — корректно отображается на десктопе, планшете и мобильном
- **Корзина на localStorage** — сохраняется между страницами и сессиями
- **Категории товаров** — фильтрация по полу (для женщин / для мужчин)
- **Уведомления** — всплывающее сообщение при добавлении товара в корзину
- **Счётчик на корзине** — показывает количество товаров
- **Мобильное меню** — бургер-меню для мобильных устройств
- **Без регистрации** — покупка без создания аккаунта

## Чего нет (по ТЗ)

- Регистрация и личный кабинет
- Онлайн-оплата
- Отзывы

## Как запустить

Просто откройте `index.html` в браузере. Сервер не требуется — это статический сайт.

```
file:///путь/к/папке/my-first-project/index.html
```
